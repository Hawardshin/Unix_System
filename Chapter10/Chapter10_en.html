Chapter 10. Learning Internet and communication commands

Direct communication between users

write

  • write username [terminal name]
  • Send a message to the logged-in user.
  • Transfer: Ctrl + D
  • Terminal name: When logging in with the same user ID several times, the terminal name may be designated.

wall

  • wall [ filename ]
  • Sends a message to all users who are logged in to the system
  • File name: Specifies when the message is written as a file in advance.
  • Normal users must belong to a tty group to use the command.
  • PATH = $PATH:/usr/sbin to use it

talk

  • talk user address [terminal name]
  • Talk 1:1 with other users
  • You can talk to users on other systems as well as on the same system
  • cctrl + c can get out.
  • User address
    • For the same system: User name
    • For other systems: username@hostname
  • If you can't talk: You need to activate the talk service!!

mesg

  • mesg [n|y]
  • Allow/prohibit output of write, wall, and talk messages to the device
  • Option:
    • y: Allow message reception
    • n: Refuse to receive messages

mailx

  • Mail server: A program that provides mail services or the system on which the program is running (POP3, SMTP)
  • Mail client: Programs that request mail services (mailx, mail, Outlook Express)
  • Mail service: /var/mail/user1
  • mailx [option] email address
    • Li? is the default promo port.
  • Reading mail is stored in the home directory mbox file.
  • Default mailbox for receiving mail: /var/mail/username
  • Option: -s: Title

finger

  • finger [username | username @hostname] | @hostname]
  • Output of logged-in user information,
  • In addition to the results of who, output additional information such as the user's full name (/etc/passwd), host name, etc.
  • Printing user's detailed personal information
  • Outputs user information logged into another host.
  • options
    • User name: Output specified user information
    • Username@Host: Output of specific user information for the specified host.
    • @Host: Output user list for specified host

Other commands

  • hostname: Output the hostname of the current system
  • nslookup [domain name]: Interactively query the domain name or IP address with the DNS server and receive a response. (Termination: exit)
  • ping: ping hostname or IP addressCheck if the system is connected over the network (although recently restricted by security)

ftp

  • ftp hostname
  • file transfer protocol
  • Command used to upload/download files to other servers over the Internet
  • Available only when ftp service is enabled

Anonymous ftp

  • The ftp server that allows you to send and receive files without an account on that server
  • Available only if provided by the host

Transmission mode

  • ASCII mode: plain text file
  • Binary mode: Images, videos, compressed files, etc.

Internal Command

  • cd: Move server directory
  • lcd:moving client directory
  • ls: Server Directory Contents
  • prompt: Do not ask y/n when sending multiple messages
  • ascii:ascii file transfer mode
  • bin: Binary transmission mode
  • get/get: Download files (multiple files)
  • put/mput: Upload files (multiple files)
  • dir: Server directory contents
  • pwd: Check server directory location
  • !pwd: Check local directory
