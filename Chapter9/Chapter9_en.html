File Archive: tar

A tar(tape archive) is a group of directories that are linked together. It is not compressed.

Option

  • tar cvf.my.tar: compress all folders under the current directory (create archive)
  • tar xvf my.tar: decompression
  • tar tvf my.tar: You can view the list.
  • uvf:update
  • rvf: Adding files

gzip

gzip has an extension.Create a compressed file that is gz. The compression rate is higher.

Option

  • -d options: decompress (same as gunzip)
  • -l : Output contents of compressed file
  • -r : Compress all directories from current directory to subdirectories
  • -t : Check compressed files
  • -v: Output to compression information screen
  • -g: Compression as much as possible

Zip

Zip creates a compressed file that is .gz.

Option

  • -v: Output to compression information screen
  • -r : Directory compression
  • -u : Outputs only modified or added files

bzip2

Compared to gzip and zip, bzip2 has a higher compression rate and slightly slower speed. Create a .bz2 file.

Option

  • -d : Unzip the file
  • -l : Shows the contents of the compressed file.
