Overview of Unix

History of Unix

  • 1969. Developed by AT&T Bell Labs (Developers: Ken Thompson, Dennis Ritchie)
  • Currently, UNIX is a registered trademark of "The Open Group", so each manufacturer uses its own product name

Linux?

  • Unix-like operating systems compatible with Unix
  • Open S/W from the beginning of development
  • 1991. Developed by Linus Torvalds
  • Torvalds currently mediates the development of the Linux kernel

The Characteristics of Unix

  • an interactive system
  • Multi-user systems / multi-task systems
  • High portability and scalability
  • Hierarchical tree file system
  • Offers a variety of additional features

Unix structure

  1. Kernel
    • The core of the operating system
    • Computer resource management (process, memory, file system, device)
  2. SHELL
    • Provide a user interface
    • Receive the user's command, process it, and output the result
  3. Utility
    • Unix System refers to a computer running Unix.
    • What you need to access the Unix system: terminal, user account
    • Terminal: Putty, HyperTerminal, Telnet, SSH (Resolving Telnet Security Issues)

Command Line Operation

Edit Command Line

  • Clear Character: Backspace
  • Erasing words: Ctrl + W
  • Erase sentence: Ctrl + U
  • Short break: Ctrl +s
  • Restart: Ctrl + q

Structure of commands

command [option] [factor]
  • Command: The program name that instructs Unix to work is divided into shell built-in commands and utilities.
  • Option: Select the detailed function of the command. It usually starts with "-" and uses numbers, uppercase letters, and lowercase letters.
  • factor: The value passed by the command, usually a file name or directory name.
  • Unix commands are case sensitive.
  • The first $ signifies prompot.

Basic command

  • banner: Print large characters on the screen - How to use: 'banner [wanted character]`
  • date: Output time and date
  • clear: Erase the screen (ins window cls)
  • man: Command description output
  • passwd: Change password (specify with 8 or more digits or special characters) (Administrators can only reset)
  • logout, Exit, Ctrl + D command: command to exit and log out the shell
  • ping 10.10.54.75: A command that sends a packet of a certain size to a computer with a target ip and receives a response from the target computer to determine network operation and status ('ping [google.co.kr ] (http://google.co.kr )'.
  • ipconfig command: command to view addresses for DNS, IP Address, Subnet Mask, Gateway (in Linux family, it is named ifconfig)
  • telnet 10.10.54.75 : Telnet to 10.10.54.75 ip address.
  • df-l-k:check partition space split command

who

The who command reads the currently accessed user information from the '/var/run/utmp' file. The utmp file stores the user's information when the user logs into the server remotely and removes the stored information when the user logs out of the remote host.
It outputs the login name, terminal, login time, remote host, or X display of the user currently connected to the server.
