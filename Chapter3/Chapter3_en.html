Option

Command

tail

  • tail [option] file name
  • Last part of the file output

Option

  • + Row number: Output from specified row to end
  • -Number: Number of rows to print on the screen
  • -f: File output is not terminated and is repeated periodically

Cp

  • cp [option] Copy original file name File name
  • Copy certain files or directories to a different name

Option

  • -i : Outputs a warning message when overwriting a file
  • -r: Copy Directory
Usage format 동작
cp file1 file2 Copy File 1 to File 2
cp files directory Copy files under the same name under the directory
cp-r directory1 directory2 Directory 1 is copied to directory 2 all the way to files in directory 1.

rm

  • rm [option] file name/directory name
  • Delete the specified file or directory
  • Files deleted once cannot be revived!
  • `rmold': If there is only one link, it will be sent to free pool, but if there are multiple links, only the file will be sent, and the inode and data block will remain the same.

Option

  • -i: Check and delete
  • -f: Delete it unconditionally
  • -r: Delete Directory

  • Rename or move files

Option

touch

  • touch [option] file
  • Create a specified file if it does not exist, change it to the current time or to the specified time

Option

  • -a: Change access time only
  • -c: Do not create a new file if it does not exist
  • -m: Change the modification time only
  • -rref_file: Read time from file
