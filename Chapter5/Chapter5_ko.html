Use Bash and SHELL

셸이란

  • 사용자와 커널 사이의 중간역활로 입력한 명령을 처리하고 실행결과를 알려준다.
  • 프롬프트 : 명령 입력을 기다리는 시스템의 상태 표시
  • 프롬포트 설정 : $PS1을 보면 된다.
  • 표준 입출력 장치 : 표준입력장치 (키보드) → 쉘 → 모니터 (표준출력, 표준 오류)
  • 파일디스크립터 : 입출력 장치 변경할 때 사용 : 0,1,2 (표준입력, 출력, 오류)

쉘의 종류

  • 본쉘 (sh)($), 콘쉘(ksh)($), C셸 (csh)(%), 배시 셸 (bash)($) 등
  • 종류는 $SHELL 환경변수를 보면 된다.
  • 로그인쉘 : 사용자가 로그인 직후 자동 생성되는 셸
  • 서브셀 : 사용자가 직접 실행한 쉘

명령

  • echo : 쉘 명령과 유틸리티 모두 제공하는 출력 명령
  • print: 쉘 명령, 옵션이 있어서 echo 보다 편리

쉘 특수문자

*특수문자 (모든의 의미)

  • ls h* : h로 시작하는 모든 파일을 보여준다. 여기엔 h도 포함
  • ls h*d : h로 시작하고 d로 끝나는 모든파일 출력

? 특수문자

  • 하나의 임의의 한문자 나타내는데 사용
  • (ls hello?.c 명령을 하면 helloA.c이런식으로 찾지만 hello.c는 포함하지 않는다.)(문자 하나만을 의미)

[ ] 특수문자

  • []안에 포함된 문자중에 하나를 나타낸다.
  • ls hello[123].c 이러면 hello1,2,3을 다 찾는다.
  • ls hello[1-3].c 와 같은의미
  • ls [A-Za-z]*[0-9] : 영문자로 시작하고 숫자로 끝나는 모든파일 출력

기타 특수문자

  • -와 ~ : 디렉토리를 나타내는 특수문자 (이전 디렉토리/ 홈)
  • ; 와 | (파이프와 ; ) : 명령과 명령을 연결하는 특수문자.
  • ‘’ 와 “”(큰따옴표 작은따옴표) : 문자열 나타내며, 작은 따옴표는 해석 을 안 하고, 큰따옴표는 $ ` \ 을 해석한다.
  • ` 특수 문자 : 명령 실행 결과를 문자열로 받고 싶을 때 사용
  • ＼(\) : 역슬레시 특수문자로 특수문자를 일반문자로 바꿈
  • >, >>, < : 입출력 방향 바꾸기 (> : 덮어쓰기 , >> : 추가하기) (명령 2>파일명 : 표준 오류 파일에 저장), (명령 < 파일명 입력을 파일로 바꾸기) ,

배쉬쉘의 변수

  • 변수 : 시스템이나 사용자에 의해 사용되는 정보를 저장하기 위한 저장소
  • 쉘변수 : 현재 쉘에서만 사용가능 변수 (확인 명령 set) (변수 = 값)
  • 환경변수: 모든 쉘에서 사용가능변수 (확인 명령 env) (export 변수=값)
  • 변수 해제 unset

인터럽트

  • ctrl + c : 현재 명령 중지
  • ctrl + D : eof
  • ctrl + \ : quit (현재 명령 중지)
  • ctrl + S : stop (화면출력 정지)
  • ctrl + Q : start (화면출력 재시작)
  • ctrl + ? : erase (마지막 문자 삭제)
  • ctrl + U : kill (전체 명령행 삭제)
  • ctrl + W : 마지막 단어 삭제
  • ctrl + Z : 명령 일시중단

나만의 명령 만들기 alias

  • alias 별칭=‘명령’
  • 여러개의 명령을 하나의 명령으로 대체가능
  • 인자가 필요한 명령에, 인자를 넘겨주지는 못함
  • 해제 : unalias 별칭 하면 된다.

이전 입력 명령 사용 fc

이전에 사용한 명령 보기 : fc ( fc -l : 라인으로 보기)

명령 재실행하기 : !

  • 명령 편집하기 : 화살표 키, 또는 Esc + k
  • 명령 탐색하기 : Esc + /
  • 여기서 마지막은 가장 최근을 의미한다.
명령 설명
!! 마지막 명령실행
!n n번째 명령 실행
!-n 현재 명령에서 뒤로 n 번째 명령 실행
!문자열 주어진 문자열로 시작하는 명령중 마지막 명령실행
<이벤트 명령> : p !! !n등 이벤트 지정자로 지정 명령 실행 안하고 화면에 출력
^문자열1^문자열2 마지막 명령의 문자열1을 문자열2로 대체하여 실행

환경 설정 파일

  • 사용자가 로그인 할 때마다 자동으로 실행되는 명령을 저장
  • 시스템 초기화 파일(/etc/profile, /etc/.login)
  • 사용자 초기화 파일 ($HOME/.bashrc, $HOME/.profile 등등)
  • 사용자 초기화 파일 실행하기 : source 명령어
