Chapter7 Searching for files and directories

grep command

  • grep [option] pattern file names
  • Search for specific strings (patterns) in the specified file
  • Pattern: Characters, strings, sentences, regular expression (/li>)
  • egrep: grep command using extended regular expression
  • fgrep: string search, interpreting all characters in a string as regular characters Disable regular expressions Literally search for all regular expressions special characters

Option

option fuction
-i Search case-insensitive
-l output file name containing the pattern
-n Each line's number is also output
-v Output lines that do not match the specified pattern
-c Output of the number of lines that match the pattern
-w Search only for patterns in one word

Regular expression

A formal language used to represent a set of strings with a particular rule

Component

  • Anchor: Express the position of the pattern in one line when searching (^, $)
  • Character Set: Express one or more characters (alphabet, number, .,[],...)
  • Modifier: Specify the number of iterations of the previous character set (*)

Special Character

character mean example result
^ Beginning of line ^String All rows starting with a string
$ End of line 'string $' All rows ending in a string
. One letter ‘a…b’ One character correspondence, searching for five characters starting with a and ending with b
[ ] One of the letters in parentheses ‘[Pp]attern’ Searching for a line that shows a pattern or pattern
[^] Non-character in parentheses ‘[^a-m]att’ Search for lines that do not appear from a to m before att
* No preceding entry or repeat multiple times ‘ab*’ Search for lines that do not have b after a or appear repeatedly
Make sure to wrap the pattern with 'minor quotation marks'!

find

  • Used by users to find specific files that exist on the system
  • Specify search scope in directory units
  • You can search by combining the properties of a file, such as its name, owner, access, and creation date.
  • Can print the absolute path of a file or execute certain commands for a file that matches the expression
  • find path search conditions [action]
    • Path: Absolute or relative path to the directory where you want to find the file
    • Search conditions: Search criteria for finding files
    • Operation: Specify the action to be performed after finding the file location, and the default action is to output the absolute path to the screen
