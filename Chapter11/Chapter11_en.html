Collection of file information

wc

  • wc [option] filename
  • Output of lines, words, and characters in the file

Option

  • -c: Byte count
  • -m: Number of characters
  • -C: -m과 동일
  • -l: Number of lines
  • -w: The number of words separated by white space or new lines

sort

  • sort [option] filename
  • Sort the contents of the text file in the way you specify and output it to the screen
  • Sort by: Environment variable: LC_COLLATE (or LANG)

Option

  • -b: Ignore the preceding space
  • -c: Output out of alignment
  • -r: Sort in reverse order
  • -u: Delete duplicate row
  • -d: sort dictionary
  • -t characters: Use the specified characters as field delimiters
  • +number: Sort by number + 1 field
  • -f: Case insensitive
  • +pos1-pos2: Specify the field to be sorted (sort from pos1 to pos2) (sort +1-2 test_sort2)

Other commands

  • split: Split a large file into several smaller files of a certain size, save standard entries in segments if no file is specified
  • uniq: If there are duplicates in a row, only one is left
  • cut: Cut selected fields from each row of the file
  • paste: pastes the contents of the specified file
  • dd: Convert the specified input file according to the specified options and save it as an output file
  • diff: command to compare differences
