파일 정보 수집

wc

  • wc [ 옵션 ] 파일명
  • 파일의 라인수, 단어수, 문자수 출력

옵션

  • -c: 바이트 수
  • -m: 문자수
  • -C: -m과 동일
  • -l: 라인수
  • -w: 화이트 스페이스나 새로운 행으로 구분된 단어의 수

sort

  • sort [옵션] 파일명
  • 텍스트 파일의 내용을 지정한 방법으로 정렬하여 화면에 출력
  • 정렬 기준: 환경변수: LC_COLLATE(또는 LANG)

옵션

  • -b: 앞에 붙는 공백을 무시
  • -c: 정렬이 되지 않은 상태로 출력
  • -r: 역순으로 정렬
  • -u: 중복행 삭제
  • -d: 사전 순 정렬
  • -t문자: 지정한 문자를 필드 구분자로 사용
  • +번호: 번호 + 1 필드를 기준으로 정렬
  • -f: 대소문자 구분 안 함
  • +pos1 -pos2: 정렬하고자 하는 필드를 지정 (pos1열부터 pos2열까지 정렬) (sort +1 -2 test_sort2)

기타 명령어

  • split: 큰 파일을 일정한 크기의 여러개 작은 파일로 분할, 파일을 지정하지 않으면 표준 입력내용을 분할 저장
  • uniq: 중복된 내용이 연속으로 있으면 하나만 남김
  • cut: 파일의 각 행에서 선택된 필드를 잘라냄
  • paste: 지정한 파일의 내용을 붙힘
  • dd: 지정한 입력 파일을 지정한 옵션에 따라 변환하여 출력 파일로 저장
  • diff: 다른점 비교하는 명령어
